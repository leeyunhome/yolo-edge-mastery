# Phase 1 — YOLO 기초

## 목표
YOLOv8~v11 학습·검증·추론 전 과정을 직접 실행하고, 버전별 아키텍처 변화를 정리한다.

## 체크리스트
- [ ] YOLOv8n/s/m COCO 사전학습 모델 추론 실행
- [ ] 커스텀 데이터셋으로 YOLOv8 학습 (Ultralytics CLI + Python API)
- [ ] v8 → v9 → v11 아키텍처 변화 정리 (C2f, RepNCSPELAN 등)
- [ ] mAP, FPS, 파라미터 수 기준 모델 크기별 비교표 작성

## 아키텍처 정리
| 버전 | 핵심 변경사항 | 비고 |
|------|--------------|------|
| v8 | C2f 블록 도입 | 기준선 |
| v9 | GELAN, PGI | 정보 손실 최소화 |
| v11 | C3k2, PSA | 경량화·정확도 균형 |
